SidePanelPane

SidePanelPane

Represents a pane within the ol-side-panel control.

Observable Properties

Name Type Description
name String | external:Element Name of this pane - User-facing name of this pane.
icon String | external:Element Icon of this pane - User-facing icon for this pane.
weight Number Weight of this pane - controls the order of the pane tabs with lower numbers appearing closer to the top and higher numbers appearing farther down. Panes with a weight >= 100 will have their icons appear at the bottom of the tab bar.
active boolean State of whether this pane is currently active.
